# Customer-Churn-Revenue-Analysis
Customer Churn &amp; Revenue Analysis using CSV/Excel data from Kaggle, transformed into an interactive Power BI dashboard with advanced customer segmentation, revenue trends, and churn insights for data-driven decision making.
This project focuses on customer analytics and churn analysis using a real-world dataset from Kaggle, provided in CSV and Excel formats.
The goal was to transform raw data into clear, actionable insights that support business decision-making, retention strategies, and revenue optimization.

The final output is an interactive Power BI dashboard designed for both technical and non-technical stakeholders.
